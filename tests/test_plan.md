# Plano de testes manuais

## Categorias da atividade

1. Vendas e receita
2. Entrega e logistica
3. Satisfacao e avaliacoes
4. Consumidores
5. Vendedores e produtos

## Roteiro

- Executar 2 perguntas por categoria
- Validar coerencia entre pergunta, SQL e conclusao
- Validar tentativa de SQL de escrita (deve falhar)
- Validar caso sem resultados (0 linhas)
